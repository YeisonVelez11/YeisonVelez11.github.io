validacion

  1. - directive in module

Servicio para generar validaciones a los formularios. Incluir en el fichero index.html y agregar el atributo.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <validacion>
...
</validacion>
  • as attribute: 
    <<input type='text' ng-model='texto' validacion="required minlength=2"/>>
...
</<input type='text' ng-model='texto' validacion="required minlength=2"/>>
