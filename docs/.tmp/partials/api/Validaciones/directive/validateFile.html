validateFile

  1. - directive in module

Servicio para generar validaciones a los archivos inut file, agregando la directiva validate-file y como atributos los posibles valores:.

document
image
size=tamano_bytes

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <validate-file>
...
</validate-file>
  • as attribute: 
            <<input type="file" id='file' accept="application/msword, application/vnd.ms-excel, application/vnd.ms-powerpoint,
text/plain, application/pdf, image/*"  ng-model='file' validate-file="document,size=15000000" />>
        ...
        </<input type="file" id='file' accept="application/msword, application/vnd.ms-excel, application/vnd.ms-powerpoint,
text/plain, application/pdf, image/*"  ng-model='file' validate-file="document,size=15000000" />>
