Servicio para generar validaciones a los formularios.

Methods

  • fn_getCamposEditables(array, data);

    Se validan individualmente los campos para la libreria de la edición de campos en la misma. Se debe agregar un atributo llamado validación en el campo a editar, luego se llama este factory
    posibles valores: 

        required
    
    minlength=number
    
    maxlength=number
    
    date
    
    email
    
    numeric
    
    minlength
    
    equalTo=string|number
    range=numer1-number2
    domain_email=domain ej bios.co
    se debe incluir en el campo los siguientes atributos ej: 
    <a  href='#' editable-text='user.name' e-label='User Name' onbeforesave='validacion.fn_getCamposEditables(this['$editable'].attrs.validacion, $data)'' validacion='required'>{{user.name}}
    donde 
    validacion='required'
    corresponde a los elementos a validar, y 
    validacion.fn_getCamposEditables
    en primera instancia 'validacion' es una variable $scope que contiene el factory en el controller(esto solo para los campos dinámicos editables), y se llama la función
    Para validar un formulario, es necesario la inyección de la directiva correspondiente.
    Es indispensable tener todo esto dentro de una etiqueta form y dentro del botón que ejecuta la función o el submit del formulario.. 
    ng-disabled="nombre_formulario.$invalid"

    Parameters

    Param Type Details
    array Array

    contiene el array de las validaciones que son puestas en el atributo "validacion".
    data String

    Corresponde al string a evaluar.

    Returns

    string

    mensaje de validación en caso de error o true en caso de estar correcto
